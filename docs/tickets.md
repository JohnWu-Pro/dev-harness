# Overview
This file defines and tracks tickets (including user stories, technical stories, and defects).

# Ticket Status and Flow
* User Story
  * [Draft] -> [Defined] -> [In-Progress] -> [Completed]
  * [Draft]/[Defined]/[In-Progress] -> [Cancelled]
* Technical Story
  * [Draft] -> [Defined] -> [In-Progress] -> [Completed]
  * [Draft]/[Defined]/[In-Progress] -> [Cancelled]
* Defect
  * [New] -> [In-Progress] -> [Fixed]
  * [New]/[In-Progress] -> [Cancelled]
